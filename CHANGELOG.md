## 0.0.1

* Initial release
* Support for BLE device discovery and connection
* Support for SoftAP (Wi-Fi) transport
* Multiple security modes: unsecured, secure (Security 1), and secure2 (Security 2)
* QR code scanning for device provisioning (iOS native, Android fallback)
* Wi-Fi network scanning from connected devices
* Device provisioning with Wi-Fi credentials
* Real-time connection and provisioning status streams
* Permission management for BLE and Wi-Fi operations
* Cross-platform support (Android & iOS)
